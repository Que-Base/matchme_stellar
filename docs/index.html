Features

Everything you need to connect.

A full-featured social app with a blockchain layer that users actually own.

🔐

Non-Custodial Wallet

Every user gets a Stellar Ed25519 keypair at signup. Secret seed lives in iOS Keychain — MatchMe never touches it.

⚡

Instant Payments

Send XLM or MATCH tokens peer-to-peer in seconds. Stellar settles in 3–5s with near-zero fees (~$0.00001/tx).

🧭

Discover & Explore

Intuitive discovery interface to find and connect with other users, backed by Firebase Firestore.

💬

Real-Time Chat

Dedicated messaging interface with likes feed and real-time interactions.

🏅

On-Chain Badges

Achievement badges minted as Soroban NFTs. Owned by the user's Stellar account — portable forever.

🛡

Verified Dates

Escrow smart contract where both users stake XLM before a date. Mutual confirmation returns stakes. No-shows forfeit.
